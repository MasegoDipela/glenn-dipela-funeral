# Programme PDF

Place the official order-of-service PDF here as:

`joshua-glenn-dipela-order-of-service.pdf`

It will be downloadable from the site via every "Download Programme" / "Download PDF" button.

If the file is missing, the buttons will still render but the download will fail. The site shows a fallback message:

> If the download does not start, please ask a family representative for the printed programme.
